Project Overview:

A simple Python program that simulates a random coin toss.

Features:

1. Randomly generates Heads or Tails

2. Allows repeated simulations

3. Demonstrates probability concepts

Concepts Used:

1. Random module

2. Conditional statements

3. Loops

How to Run:

1. Run the script.

2. The program randomly displays Heads or Tails.

3. Repeat as desired.

Learning Outcome:

This project helped develop understanding of randomness and probability simulation using Python.
